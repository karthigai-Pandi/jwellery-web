# Jewellery Landing Page

A polished static landing page inspired by the Jos Alukkas online storefront.

## Files

- `index.html` — page markup
- `styles.css` — responsive styling
- `script.js` — interactive ticker, mobile menu, and search demo

## Usage

Open `index.html` in a browser to view the website. For local development, you can also use a simple static server.

## Notes

- The page is fully responsive and includes mobile navigation.
- The search form uses a demo handler and does not connect to a backend.
- The WhatsApp button opens support chat externally.
